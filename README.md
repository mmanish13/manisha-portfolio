# Manisha — Portfolio

Senior Full Stack Developer specializing in .NET, Angular, Azure, and building scalable enterprise systems for healthcare, life sciences, and financial services.

🌐 **Live Demo:** [manisha-portfolio-two.vercel.app](https://manisha-portfolio-two.vercel.app/)

Built with **Angular 19** (standalone components) + **Tailwind CSS v3**.

## Quick Start

```bash
ng serve
```

Navigate to `http://localhost:4200/`.

## Build

```bash
ng build
```

Build artifacts are output to `dist/`.
